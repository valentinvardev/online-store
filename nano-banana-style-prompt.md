# Nano Banana 2 — Style Prompt: La Reina de Bastos

> Usá este prompt ANTES de describir la ilustración específica.
> Define el estilo, paleta y mood para toda la serie de ilustraciones del sitio.

---

## Prompt base (en inglés, para mejores resultados)

```
Illustration style: 1960s–1970s psychedelic folk art, flower power, groovy bohemian.
Flat design with bold outlines. Warm, rich, fully saturated colors — no pastels, no muted tones.
Organic shapes, hand-drawn feel, decorative and festive.

Color palette — use these exact tones:
- Forest green: #3D7A47 and #6BAE75
- Sky blue: #7EC8E3 and #B0E0F5
- Sunflower yellow / gold: #F5C842 and #FFE066
- Warm orange / amber: #E8845A and #FF9A5C
- Psychedelic pink / fuchsia: #D45FA0 and #F07DC0
- Purple / violet: #7B5EA7 and #A882D8
- Cream / bone (background): #FBF5E6
- Earth brown: #8B5E3C

Decorative elements to include when fitting: geometric florals, mandalas, tropical leaves,
sunbursts, crescent moons, stars, concentric rings, wavy lines, symmetrical botanical motifs.

Texture: slight grain or paper texture, watercolor wash feeling, organic and warm.
NOT digital-clean, NOT minimalist, NOT thin lines, NOT cold colors, NOT generic new-age white-background spirituality.

The mood is: earthy, festive, sensual, mystical, joyful — like a wise friend who also knows how to dance.
Queen of Wands energy: confident, creative, connected to nature, spiritually alive.
```

## Cómo pedir un elemento decorativo aislado (para recortar el fondo)

> Este es el formato exacto para generar una pieza suelta — luna, flor, cristal, lo que sea.
> El objetivo: fondo sólido de un solo color, elemento centrado, listo para hacer remove background.

### Estructura del prompt completo

```
[PEGAR PROMPT BASE ARRIBA]

Now generate a single decorative element: [DESCRIBIR EL ELEMENTO].
The element must be centered in a square 1:1 composition.
Solid flat background — use cream #FBF5E6 (no gradients, no patterns, no texture on the background).
Nothing else in the image. No text, no border, no frame, no shadows that extend to the edges.
The element should be bold, detailed, and fill roughly 70–80% of the frame.
```

### Ejemplos listos para copiar

**Luna creciente:**
```
Single decorative element: a crescent moon with a face, surrounded by small stars and swirling lines.
Centered, 1:1 square format. Solid background color #FBF5E6. Nothing else in the image.
Element fills 70% of the frame. Bold outlines, rich colors from the palette.
```

**Flor psicodélica:**
```
Single decorative element: a large psychedelic flower with concentric petals in alternating colors.
Centered, 1:1 square format. Solid background color #FBF5E6. Nothing else in the image.
Element fills 75% of the frame. Bold flat outlines, groovy 70s style.
```

**Cristal / gema:**
```
Single decorative element: a faceted crystal gem with geometric shine lines radiating outward.
Centered, 1:1 square format. Solid background color #FBF5E6. Nothing else in the image.
Element fills 65% of the frame. Bold outlines, mystical feel.
```

### Negative prompt (si el generador lo permite)

```
background pattern, texture behind element, gradient background, border, frame, shadow, text, watermark, multiple elements, busy composition, white background, minimalist
```

---

## Notas de uso

- **Pegá el prompt base primero**, luego en el mismo campo (o en el siguiente) describí la ilustración específica.
- Si Nano Banana tiene campo de "negative prompt", agregá: `minimalist, thin lines, white background, cold colors, digital clean, generic spiritual aesthetic, watermark`
- Para consistencia entre ilustraciones, mantené la misma seed o estilo guardado entre generaciones de la misma sesión.
- El fondo ideal para todas las piezas es **crema #FBF5E6** o transparente, para que encaje con el sitio.
